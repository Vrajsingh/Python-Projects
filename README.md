# Python-Projects
## INTRODUCTION - SYSTEM SETUP

### PYTHON INSTALLATION - LATEST VERSION

STEP 1 : Download and Install Latest Version of Python : [LINK](https://www.python.org/downloads/)

STEP 2 : Open Terminal/cmd and type "python --version" 
// You must get the installed python version as output

STEP 3 : Type "pip --version"
// pip is the standard package manager for Python. It allows you to install and manage additional packages that are not part of the Python standard library. 
// pip comes as default with python installation but in case you get an error then type and run "sudo easy_install pip"

### VISUAL STUDIO CODE INSTALLATION - IDE

STEP 1 : Download and Install Latest Version of Visual Studio Code : [LINK](https://code.visualstudio.com/)

STEP 2 : After successfull installation, Click on windows search bar and type "Visual Studio Code". You will see the VS Code App, open it.

//Now Configure VS Code for running python program
STEP 3 : Click on Extension button and in search bar type "Code Runner" and install it.

//Install Python for VS Code
STEP 4 : In the extension search bar only type "python" and select first option and install Python (Make sure it is from MICROSOFT).

//Test python code
STEP 5 : Create new file with ".py" extension and write basic python code " print("Hello World") " and run the code by clicking on Run button available on extreme right

//Configure Code Runner Run in Terminal
STEP 6 : Open File -> Preferences -> Settings -> Type "run in terminal" -> Check the box "Code runner Run in Terminal"

STEP 7 : Restart VS Code and Congratulation Your System is Ready Now !!
